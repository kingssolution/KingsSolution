# KingsSolution

## Overview
KingsSolution is an online educational platform built to support Nigerian students preparing for WAEC, NECO, JAMB, and other examinations. The platform delivers verified exam updates, learning resources, and structured academic guidance to help students prepare effectively.

---

## Features
- Latest exam news and official updates  
- Verified educational content reviewed before publishing  
- Subject-based learning materials (Mathematics, English, Sciences, etc.)  
- Mobile-friendly and student-focused interface  
- Secure authentication system  
- Admin dashboard for content moderation and management  

---

## Technologies Used
- Frontend: HTML5, CSS3, JavaScript  
- Backend: Firebase (Firestore & Authentication)  
- Image Hosting: Cloudinary  
- Hosting: Netlify  

---

## How It Works
1. Students sign up or log in to access available content  
2. Users browse subjects or exam categories (WAEC, NECO, JAMB, etc.)  
3. Content is displayed in text and image formats where applicable  
4. Admins review and manage posts, users, and updates via a dashboard

## Setup & Installation
1. Clone the repository  
   git clone https://github.com/ki-ng0/Kingssolution.git  

2. Navigate into the project  
   cd Kingssolution  

3. Configure Firebase  
   - Create a Firebase project  
   - Enable Firestore and Authentication  
   - Replace Firebase configuration values in the project  

4. Run locally  
   Open index.html in your browser  
   (or use Live Server in VS Code)
